Why TikTok and Meta ROAS aren't directly comparable

Each platform uses a different default attribution window and counting methodology. Meta's 7-day click + 1-day view window credits far more assists than TikTok's default 7-day click. A direct ROAS comparison using each platform's reported numbers will almost always make Meta look stronger — even when TikTok is delivering better incremental revenue.

How to normalise for a true comparison

Normalising both platforms to a 1-day click attribution window (the most conservative and most comparable window available on both platforms) gives the closest approximation of true incremental performance. This calculator applies a platform-specific deflation factor to each platform's reported ROAS to estimate normalised performance.

Which platform wins by product category

Fashion, beauty, food, and impulse-buy products consistently show stronger TikTok normalised ROAS — the visual-first format matches the discovery mindset. Higher-consideration purchases (furniture, electronics, B2B) typically perform better on Meta, where users arrive with more purchase intent.
